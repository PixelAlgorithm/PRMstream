# PRMStream

PRMStream is a modern streaming application featuring a full-stack architecture with a Web Frontend and a Node.js Backend. It leverages tools like React, Supabase, and various other technologies to deliver a seamless streaming experience.

> **⚠️ Disclaimer: Educational Purpose Only**
> 
> I **do not condone piracy**. This application is strictly an **educational project** built to learn full-stack development and modern UI/UX design patterns. The application aggregates and uses third-party streaming services and APIs to fetch content. No proprietary content is hosted on these servers.

---

## 📸 Features

- **TV & Movie Streaming**: Fluid interface for web browsers.
- **Dark Mode / Light Mode**: Dynamic theming support.
- **Secure Authentication**: Handled via Supabase logic.
- **Responsive Design**: Optimized for viewing on desktops, tablets, and mobile browsers.

---

## 🛠️ Tech Stack

### Frontend (Web)
- **Framework**: React / Vite
- **Styling**: Vanilla CSS / Modules
- **Hosting**: Vercel

### Backend & Database
- **Server**: Node.js & Express
- **Database / Auth**: Supabase (PostgreSQL)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed along with `npm` or `yarn`. 

### 1. Backend Setup
1. Navigate to the `backend/` directory.
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and fill in your Supabase variables.
4. Run the server: `npm run dev` or `node server.js`

### 2. Frontend (Web) Setup
1. Navigate to the `frontend/PRMStream/` directory.
2. Install dependencies: `npm install`
3. Set your environment variables (pointing to your backend/Supabase).
4. Run the dev server: `npm run dev`
